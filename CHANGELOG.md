## [1.0.0] — 2026-05-09

### Added
- Initial repository created at [github.com/4thandBailey/bimi](https://github.com/4thandBailey/bimi)
- `logo.svg` — 4th and Bailey brand logo published in SVG Tiny P/S format (256×256px)
- `README.md` — Repository documentation
- GitHub Pages enabled to serve `logo.svg` over HTTPS at `https://4thandbailey.github.io/bimi/logo.svg`
- BIMI DNS TXT record published for `4thandbailey.com`
- BIMI DNS TXT record published for `4thandbailey.net`
- Domain verification completed for `4thandbailey.com` via GitHub Pages verified domains
- Domain verification completed for `4thandbailey.net` via GitHub Pages verified domains
- DMARC policy updated to `p=reject; sp=reject` on `4thandbailey.com`
- DMARC policy updated to `p=reject; sp=reject` on `4thandbailey.net`
- BIMI Brand Logo confirmed found via MXToolbox on both domains

### Validated
- MXToolbox BIMI Checker — `4thandbailey.com` ✅
- MXToolbox BIMI Checker — `4thandbailey.net` ✅
